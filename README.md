# ML Foundations

Your one-stop resource for the mathematical and machine learning concepts you actually need to know — from probability, Calculus and linear algebra to neural networks and beyond.

## What's Inside

- **Statistics & Probability** — distributions, hypothesis testing, Bayesian thinking
- **Linear Algebra** — vectors, matrices, eigenvalues, and the geometry behind ML
- **Calculus & Optimization** — gradients, backpropagation, convex optimization
- **Machine Learning** — regression, classification, clustering, ensemble methods
- **Deep Learning** — architectures, training dynamics, regularization
- **Math for ML** — information theory, numerical methods, and everything in between

## Status

> **Work in progress.** This repo is actively being built out. New topics and notebooks are added regularly. Contributions and suggestions are welcome!

## Why This Repo?

Most ML resources either drown you in theory with no code, or hand you code with no intuition. This repo aims to bridge both — clear explanations paired with hands-on implementations so the concepts actually stick.
